# SensyFlow Landing Page (React + Vite + Tailwind)

A high-level clone-style SaaS landing page inspired by AutoSensy/Wati/AiSensy design language:
- Dark + neon green conversational commerce theme
- Animated hero dashboard + floating chat card
- Advanced section animations and orbiting omnichannel module
- Built with **React + Vite + Tailwind CSS**

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
